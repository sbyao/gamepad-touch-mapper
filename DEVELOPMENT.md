# 开发文档

## 项目结构

```
.
├── 主程序.py              # 主应用程序入口
├── 虚拟键盘网格布局.py     # 虚拟键盘界面模块
├── README.md              # 使用文档
├── DEVELOPMENT.md         # 开发文档
├── 手柄配置.json          # 手柄映射配置文件（自动生成）
└── 按键方案.json          # 按键方案配置文件（自动生成）
```

## 核心模块

### 主程序.py

#### 主要类

**App** - 主应用程序类
- 继承自 `tk.Tk`
- 负责UI界面和事件处理
- 管理手柄输入监听线程

#### 关键函数

| 函数名 | 功能 |
|--------|------|
| `send_touch_event` | 发送触摸/点击事件到指定坐标 |
| `simulate_key` | 模拟键盘按键（支持组合键） |
| `get_window_rect` | 获取窗口位置和大小 |
| `load_config` | 加载配置文件 |
| `save_config` | 保存配置文件 |

#### 数据结构

**VK_KEYS** - 虚拟键码映射字典
```python
{
    'A': 0x41,
    'B': 0x42,
    # ... 更多按键
}
```

**KEY_DISPLAY_NAMES** - 按键显示名称映射
```python
{
    'MEDIA_PREV': '上一曲',
    'CTRL': 'Ctrl',
    # ... 更多映射
}
```

### 虚拟键盘网格布局.py

#### 主要类

**VirtualKeyboard** - 虚拟键盘弹窗类
- 继承自 `tk.Toplevel`
- 提供可视化按键选择界面
- 支持组合键配置

#### 关键方法

| 方法名 | 功能 |
|--------|------|
| `create_keyboard` | 创建键盘布局 |
| `create_button` | 创建单个按键按钮 |
| `on_key_click` | 按键点击处理 |
| `toggle_combo_mode` | 切换组合键模式 |
| `save_key_scheme` | 保存按键方案 |

#### 网格布局参数

```python
self.cell_width = 35    # 单元格宽度
self.cell_height = 35   # 单元格高度
self.gap = 2            # 间隙大小
```

## 输入处理流程

### 手柄输入监听

1. `gamepad_listener` 线程持续监听手柄事件
2. 事件放入 `event_queue` 队列
3. `process_events` 方法处理队列中的事件
4. 根据配置执行相应操作

### 事件类型处理

| 事件类型 | 处理方式 |
|----------|----------|
| 按钮按下 | 触发配置的点击或按键操作 |
| 摇杆移动 | 转换为鼠标移动或方向键 |
| 扳机按下 | 根据阈值触发操作 |

## 坐标系统

### 屏幕坐标
- 相对于整个屏幕的绝对坐标
- 原点 (0,0) 在屏幕左上角

### 窗口坐标
- 相对于目标窗口客户区的相对坐标
- 窗口移动时坐标自动跟随

## 扩展开发

### 添加新的按键映射

1. 在 `VK_KEYS` 中添加键码映射
2. 在 `KEY_DISPLAY_NAMES` 中添加显示名称
3. 更新虚拟键盘布局

### 添加新的操作类型

1. 在功能下拉框中添加新选项
2. 在 `handle_button_event` 中添加处理逻辑
3. 更新配置保存/加载逻辑

## 调试技巧

- 使用管理员权限运行以支持输入模拟
- 检查 `手柄配置.json` 确认配置已保存
- 查看控制台输出了解程序状态
