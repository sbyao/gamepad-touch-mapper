# 游戏手柄触屏映射工具

将游戏手柄按键映射为屏幕触摸操作或键盘按键的Windows应用程序。

## 功能特性

- **手柄按键映射**：支持Xbox手柄的所有按键映射
- **多种操作模式**：
  - 屏幕点击：在指定屏幕坐标执行点击
  - 窗口点击：在指定窗口相对坐标执行点击
  - 键盘按键：模拟单个按键或组合键
  - 鼠标控制：左摇杆控制鼠标移动
- **虚拟键盘**：可视化选择按键，支持组合键配置
- **方案管理**：保存和加载不同的映射方案
- **系统托盘**：最小化到系统托盘，不占用任务栏

## 安装要求

- Windows操作系统
- Python 3.7+
- 游戏手柄（Xbox手柄推荐）

## 安装步骤

1. 克隆或下载本仓库
2. 安装依赖：
   ```bash
   pip install inputs pillow pystray
   ```

## 使用方法

### 启动程序

```bash
python 主程序.py
```

### 配置映射

1. **选择目标窗口**：在"目标窗口"下拉框选择要操作的窗口
2. **配置按键**：
   - 选择功能类型（点击/按键/鼠标）
   - 输入屏幕坐标或选择按键
   - 点击"保存配置"
3. **启动映射**：点击"启动映射"按钮开始监听手柄输入

### 坐标选取

1. 点击按键名称（如"A键"）
2. 程序自动跳转到目标窗口
3. 在目标位置**再次点击鼠标左键**记录坐标
4. 坐标自动填入屏幕坐标和窗口坐标输入框

### 虚拟键盘

1. 在功能列选择"按键"
2. 点击"选按键"按钮打开虚拟键盘
3. 选择单个按键或点击"组合键"按钮进入组合键模式
4. 依次点击要组合按键
5. 点击"保存按键方案"完成配置

### 系统托盘

- **启动映射后**：程序自动最小化到系统托盘
- **右键菜单**：
  - 启动/停止映射（根据当前状态）
  - 显示窗口
  - 退出
- **双击图标**：显示主窗口

### 方案管理

- **新建方案**：点击"新建"按钮创建新方案
- **切换方案**：在"当前"下拉框选择已有方案
- **删除方案**：选择方案后点击"删除"按钮

## 配置文件

- `手柄配置.json`：存储按键映射配置
- `按键方案.json`：存储组合键方案

## 支持的按键

### 手柄按键
- A、B、X、Y按钮
- LB、RB肩键
- LT、RT扳机键
- 方向键（十字键）
- 左摇杆、右摇杆
- L3、R3摇杆按下
- Back、Start按钮

### 键盘按键
- 所有字母键（A-Z）
- 数字键（0-9）
- 功能键（F1-F12）
- 控制键（Ctrl、Alt、Shift、Win）
- 方向键
- 媒体键（音量、播放控制）
- 组合键（如 Ctrl+A、Alt+F4）

## 注意事项

- **管理员权限**：需要以管理员权限运行才能模拟鼠标/键盘输入
- **手柄连接**：确保手柄已连接并被系统识别，启动程序前请先连接手柄
- **坐标格式**：坐标值格式为 x,y（如 100,200）
- **坐标选取**：点击按键名称后，程序会跳转窗口，需要在目标位置**再次点击**才能记录坐标
- **系统托盘**：启动映射后程序自动隐藏到系统托盘，可通过右键菜单控制
- **窗口激活**：目标窗口需要处于激活状态才能接收模拟输入
- **杀毒软件**：部分杀毒软件可能拦截输入模拟，请添加信任或关闭
