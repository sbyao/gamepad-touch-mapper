# 游戏手柄触屏映射工具

将游戏手柄按键映射为屏幕触摸操作的工具，支持多种操作模式和灵活的配置方式。

## 功能特性

- **多种操作类型**: 点击、长按、上滑、下滑
- **摇杆多模式**: 方向键模式、模拟摇杆模式、鼠标模式
- **坐标配置**: 支持屏幕绝对坐标和窗口相对坐标
- **目标窗口**: 可指定特定窗口，自动激活并计算相对坐标
- **可视化界面**: 手柄布局可视化，按键状态实时显示
- **配置保存**: 自动保存配置到JSON文件

## 系统要求

- Windows 操作系统
- Python 3.7+
- 支持的游戏手柄（Xbox/PS等）

## 安装

1. 克隆或下载本项目
2. 安装依赖:
```bash
pip install inputs pillow pystray
```

## 使用方法

### 1. 启动程序

```bash
python 演示.py
```

### 2. 配置按键映射

1. **选择目标窗口**: 在"目标窗口"下拉框中选择要操作的窗口，或选择"整个屏幕"
2. **捕获坐标**: 点击按键列表中的按键名称，然后点击目标位置，程序会自动捕获屏幕坐标和窗口相对坐标
3. **选择功能**: 为每个按键选择要执行的操作（点击/长按/上滑/下滑）
4. **保存配置**: 点击"保存配置"按钮

### 3. 摇杆配置

摇杆支持三种模式:
- **方向键**: 模拟方向键滚动，适合列表浏览
- **摇杆**: 在基准坐标周围移动，适合精确控制
- **鼠标**: 控制鼠标移动，适合通用操作

### 4. 启动映射

点击"启动映射"按钮开始监听手柄输入并执行对应的屏幕操作。

| 按键代码 | 名称 |
|---------|------|
| BTN_SOUTH | A键 |
| BTN_EAST | B键 |
| BTN_NORTH | Y键 |
| BTN_WEST | X键 |

| BTN_TR | RB |

| BTN_START | Start |

| BTN_THUMBR | R3 |

| ABS_HAT0D | 十字下 |
| ABS_HAT0L | 十字左 |
| ABS_HAT0R | 十字右 |

| ABS_RIGHT_STICK | 右摇杆 |

## 配置文件

配置自动保存在 `gamepad_config.json` 文件中，格式如下:

```json
{
  "BTN_SOUTH": {
    "screen_x": "100",
    "screen_y": "200",
    "window_x": "50",
    "window_y": "100",
    "func": "点击"
  }
}
```

## 注意事项

1. 首次使用请先点击"检测手柄"确认手柄已连接
2. 使用窗口相对坐标时，目标窗口必须处于运行状态
3. 鼠标模式下摇杆有15%的死区，避免轻微漂移
4. 程序需要管理员权限才能模拟鼠标操作
